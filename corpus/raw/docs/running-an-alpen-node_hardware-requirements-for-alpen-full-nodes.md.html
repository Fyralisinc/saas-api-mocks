# Hardware requirements for Alpen full nodes To run an Alpen full node, a Windows, Linux, or Mac machine with the following minimum specifications is needed: * CPU: 64-bit processor with at least 2c4t, for better performance you may try with a higher number of CPUs * Memory: 8GB * Storage: 1TB (NVMe SSD recommended) * Network: 20 Mbps --- # Agent Instructions: Querying This Documentation If you need additional information that is not directly available in this page, you can query the documentation dynamically by asking a question.
