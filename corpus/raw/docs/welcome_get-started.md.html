# Get started Welcome to the Alpen Testnet community! To get started, here are the details you'll need: ## Wallets Alpen is an EVM-compatible blockchain that will work with EVM wallets out of the box. You can therefore use any EVM wallet that supports adding custom networks. Here are a few that are known to work: * [Enkrypt](https://www.enkrypt.com/) * [Frame](https://frame.sh/) * [MetaMask](https://metamask.io/) ## Resources {% hint style="danger" %} The Alpen testnet has been updated to the [Prague spec](/how-alpen-works/comparing-strata-and-ethereum.md). This update was implemented with a network regenesis, resulting in the reset of all state, accounts, and smart contracts. The Shanghai-based Alpen testnet (chain ID `2892`) was sunset on December 22, 2025. For more information read the [announcement blogpost](https://www.alpenlabs.io/blog). {% endhint %}
